Task:
	Некоторая программа хранит протокол работы пользователей системы в виде текстового файла. 
Данный файл содержит записи о регистрации пользователей системы следующей структуры:

login role date time,

где login и role – поля с именем (логином) и ролью (набором прав) пользователя, 
каждое из которых содержит непустую последовательность символов без пробелов,
 date – дата регистрации (входа в систему), заданное в формате число.месяц.год,
 time – время регистрации в формате час:минуты:секунды. 
Поля разделяются одним или более пробелами. 
Каждая запись о регистрации располагается в отдельной строке без переносов.

Данный файл не содержит:
•	пустых строк или строк в формате, отличающемся от представленного;
•	пустых (неопределённых) имен, ролей, дат и времён;
•	дат и времён, заданных в других форматах или содержащих неверные данные, 
например, неправильное число месяца или ошибочный час.

Также заданы две таблицы следующей структуры:
Users (Пользователи)
id integer primary key
login char (8) unique

2. Registration (Регистрация)
code integer primary key
id integer foreign key references Users(id)
role varchar(16)
date timestamp

Напишите программу, которая в существующей базе данных создает обе таблицы и заносит в эти таблицы данные, 
считанные из представленного файла с протоколом работы системы. 
Затем программа должна выполнить следующие запросы к сформированным таблицам:
•	выбрать все регистрационные записи (запрос должен точно воспроизвести исходный файл протокола);
•	выбрать всех пользователей, которые регистрировались (входили в систему) не ранее заданной даты 
(дата определяется как параметр программы или вводится пользователем в диалоговом окне);
•	выбрать всех пользователей, которые регистрировались с разными ролями (т.е. не мене чем с двумя);
•	выбрать все даты, в которые регистрировалось не менее двух пользователей.
